Structure, Phrasing and Display

In HTML, the default display value for most emelents is block or inline. Structual(block) elements will create larger structures than inline elements. By the default formatting, block elements begin on new lines, but inline elements can start content anywhere in a line. A block element always takes up the full width available on the browser. For inline element, it only takes up as much width as necessary, like the "a" and "img" tag. If we want some contents that width 100% of the browser, we should apply block element and asign CSS attribute.

Box Model

All elements in HTML can be considered as boxes. Boxes control the design and layout of a webpage. The CSS box model is essentially a box that wraps around all HTML elements. Its attributes include width, margins, borders, padding and so on. We can assign different values to these attributes and change the overall layout and typography of the page. The box-sizing property sets the total width and height of an element. By default in CSS box, the width and height we assigned to elements will be applied only to the element's content box. When we assign the width and height, we have to adjust the value for any border or padding.

Background Images

The "img" tag in HTML means the image is related to the content and is not just a design element. It should be display with the text. While the background-image in CSS means that this image is not part of the content, and it only works as a background. Also, the background-image will not be printed by default when printing and image with img tag in HTML will be printed automatically with other content.
